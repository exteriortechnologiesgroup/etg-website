// HOW WE WORK

A five-phase
engagement

// PHASE 01

Discovery

Initial consultation to understand project scope, design intent, performance requirements, and budget envelope.

// PHASE 02

Selection

Material recommendations, sample delivery, finish selection, and confirmation of compatibility with adjacent assemblies.

// PHASE 03

Specification

Spec writing assistance, AutoCAD details, BIM coordination support, and compliance documentation.

// PHASE 04

Pricing

High-level budget pricing during DD, refined to firm pricing at CDs. Value engineering options provided when required.

// PHASE 05

Field Review

On-site field review during installation, mock-up review, and post-installation verification of performance criteria.
